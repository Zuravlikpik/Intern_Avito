# API-тесты для [проекта](https://qa-internship.avito.com/)
## Запуск тестов

1. Склонируйте себе репозиторий: `git clone <путь к репозиторию>`
2. [Установите](https://go.dev/doc/install) Go версии 1.25.1
3. Подтяните необходимые зависимости через команду в корне проекта:

```bash
   go get -u -v -t all
```

4. Для запуска всех тестов перейдите в корень проекта:
```bash
   cd task_aut
```
и запустите:
```bash
  go test -v ./...
```

5. Отдельный тест можно запустить командой:

```bash
  go test <путь до go-файла с тестом>
```

## Логирование

Для включения логирования request/response в консоль: выставьте переменную окружения `DEBUG=true` в
`.env`

## Запуск линтера

1. Установить приложение [golangci-lint](https://golangci-lint.run/docs/welcome/install/local/)
2. Запуск линтера (из корня репозитория):

```bash
  golangci-lint run
```
